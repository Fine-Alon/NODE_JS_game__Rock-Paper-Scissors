# NODE_JS_game__Rock-Paper-Scissors
CLI game

🪨📄✂️ Rock, Paper, Scissors - CLI Edition
About the Project
A stylish and interactive Command Line Interface (CLI) implementation of the classic Rock, Paper, Scissors game, built with Node.js.

Unlike standard text-based terminal games, this project focuses on providing a clean, highly visual user experience directly in the console. It features custom-framed menus, a real-time updating scoreboard, and emoji integration to make the gameplay feel polished and engaging. Players can set custom match lengths and face off against a randomized bot opponent in a race to victory.

✨ Key Features
Customizable Match Length: Players can choose to play anywhere from a quick "Best of 3" up to a 10-round marathon.

Dynamic Scoreboard: A framed, real-time UI that tracks the user and bot scores cleanly throughout the game.

Polished Terminal UI: Beautifully formatted headers, round dividers, and game-over screens utilizing box-drawing characters and emojis.

Smart Tie Handling: If both players make the same choice, the game detects the draw and repeats the round without consuming one of the player's requested turns.

Input Validation: Ensures smooth gameplay by catching invalid user inputs during the round selection and weapon choice phases.

🚀 Getting Started
Prerequisites
To run this game on your local machine, you will need Node.js installed, along with the prompt-sync package for handling terminal inputs.
